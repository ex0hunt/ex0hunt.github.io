Подготовка. 

*   CONFIG_B43:         should not be set. But it is.
 *   CONFIG_SSB:         should not be set. But it is.
 *   CONFIG_MAC80211:    should not be set. But it is.
 CONFIG_B43:

       -> Device Drivers                                                                                       
         -> Network device support                                                            
           -> Wireless LAN                                                                        
             -> Broadcom 43xx-legacy wireless support (mac80211 stack)                        
             -> Broadcom 43xx-legacy data transfer mode 


 CONFIG_SSB:
Device Drivers -> Sonics Silicon Backplane

 CONFIG_MAC80211:


       -> Networking support                                                                       

         -> Wireless -> mac80211

Установка

emerge -av broadcom-sta
