--- layout: post title: Отключения микрофона date: '2014-10-20T02:42:00.004+04:00' author: Дмитрий К tags: modified_time: '2014-10-20T02:42:25.908+04:00' blogger_id: tag:blogger.com,1999:blog-3722882267608887154.post-3274370414487723320 blogger_orig_url: http://ubufang.blogspot.com/2014/10/blog-post.html --- Начиная с ядра 3.13 поддерживается кнопка отключения микрофона, которая присутствует на некоторых ноутбуках, например Thinkpad. Но, в KDE-4 механизм обработки события MICMUTE отсутствует, потому необходимо установить действие на нажатие вручную. Ниже приведены необходимые действия для ноутбуков thinkpad Вначале необходимо отредактировать файл /lib/udev/hwdb.d/60-keyboard.hwdb, заменить ########################################################### # Lenovo ########################################################### # thinkpad_acpi driver keyboard:name:ThinkPad Extra Buttons:dmi:bvn*:bvr*:bd*:svnLENOVO*:pn* KEYBOARD_KEY_1a=f20 #  

на

 ########################################################### # Lenovo ########################################################### # thinkpad_acpi driver keyboard:name:ThinkPad Extra Buttons:dmi:bvn*:bvr*:bd*:svnLENOVO*:pn* KEYBOARD_KEY_1a=prog2 # После этого выполнить
sudo udevadm hwdb --update
и сделать ре-логин с систему.&
Осталось добавить действие (KDE->Параметры системы->Специальные действия) amixer set Capture toggle на нажатие кнопки
